# Turtle-Crossing-Game
A fun arcade-style game built with Python and the Turtle graphics module. The objective is simple: help the turtle cross the road while avoiding oncoming traffic. Each successful crossing increases the game's difficulty by making the cars move faster.

Features :-

- Simple keyboard controls
- Randomly generated moving cars
- Increasing difficulty after every successful level
- Level tracking
- Game Over screen on collision
- Built using Python's built-in Turtle graphics library

How to Play :-

1. Run the Python script.

2. Press the Up Arrow key to move the turtle forward.

3. Avoid colliding with moving cars.

4. Reach the top of the screen to complete the level.

5. Each new level increases the speed of the cars.

6. The game ends when the turtle collides with a car.

Controls :-

     Key	           Action
↑ Up Arrow	Move the turtle forward

Gameplay :- 

- The turtle starts at the bottom of the screen.

- Cars continuously move across the road.

- Every successful crossing increases the level.

- Higher levels increase the speed of the cars.

- A collision ends the game.
